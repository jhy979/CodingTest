# Map (javascript)

```js
  // 선언 및 삽입
  let me = new Map();
  me.set('name', 'kevin');
  
  // 탐색 및 검색
  console.log(me.get('name'); // kevin
  
  // has(): 주어진 key 가 존재하는지 확인
  console.log(me.has('name')); // true
```
